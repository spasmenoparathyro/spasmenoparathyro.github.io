Μία ακόμα πρωτιά χάνει η Ελλάδα από τις ΗΠΑ, αυτή του πιο ανατριχιαστικού παιδικού ποιήματος. Έτσι, μετά τον Κωνσταντή και τη μάνα με τους εφτά της γιους και με τη μια της κόρη, μας έρχεται από τη Μασαχουσέτη ένα ποίημα που σκέφτηκε δάσκαλος σε νηπιαγωγείο, προκειμένου να μάθουν τα παιδιά τι πρέπει να κάνουν σε περίπτωση που κάποιος ένοπλος αρχίσει να σκοτώνει κόσμο στο σχολείο τους.

Το ποιηματάκι, πάει κάπως έτσι:

Lockdown. Lockdown
Lock the door
Shut the lights off
Say no more
Go behind the desk
and hide
Wait until it’s safe inside
Lockdown. Lockdown
it’s all done
Now it’s time to have
some fun

Με τα περιστατικά επιθέσεων σε σχολεία στις ΗΠΑ, να γίνονται πλέον μια φορά τη βδομάδα, καθηγητές και μαθητές έχουν στραφεί σε όποια μέθοδο πιστεύουν πως μπορεί να τους προστατεύσει καλύτερα, αφού η κεντρική κυβέρνηση εμφανίζεται απρόθυμη να περιορίσει την ελεύθερη διακίνηση όπλων, για χάρη των κερδών της οπλικής βιομηχανίας.

Στη Λουιζιάνα για παράδειγμα, έχουν επιτρέψει στους μαθητές να κυκλοφορούν με αλεξίσφαιρα σακίδια πλάτης.

Στην Πενσυλβάνια, έχουν δώσει αλεξίσφαιρες ασπίδες. Στην ίδια πολιτεία, μια περιφέρεια αποφάσισε να εξοπλίσει με μικρά ρόπαλα του μπέιζμπολ τους περίπου 500 δασκάλους και καθηγητές της. Οι καθηγητές παρέλαβαν τα ρόπαλά τους, στο τέλος ενός σεμιναρίου που αφορούσε τη διαχείριση ανάλογων περιστατικών.

“Είναι το τελευταίο μέσο για να καταφύγεις, αλλά είναι μια επιλογή και αυτό θέλουμε να το γνωρίζουν όλοι. Δυστυχώς είμαστε σε μια εποχή, που κάποιος μπορεί να τα χρειαστεί για να προστατεύσει τον εαυτό του και τα παιδιά μας”, δήλωσε ο επόπτης της Περιφέρεια.

Στην ίδια πολιτεία, λίγες μέρες πριν, ένα άλλο σχολείο, εξοπλίστηκε με πέτρες, για να αντιμετωπίσει τους ένοπλους. Για την ακρίβεια, σε κάθε αίθουσα, τοποθετήθηκε ένας κουβάς με πέτρες, ενώ o αντίστοιχος επόπτης είχε δηλώσει:

Αν ένας ένοπλος προσπαθήσει να μπει σε οποιαδήποτε από τις σχολικές αίθουσες, θα έρθει αντιμέτωπος με μια τάξη γεμάτη μαθητές που θα κρατάνε πέτρες και θα πετροβοληθεί.

Είχε προηγηθεί δήλωση του προέδρου των ΗΠΑ Ντόναλντ Τραμπ, μπροστά σε παιδιά που επιβίωσαν από ένα μακελειό σε σχολείο της Φλόριντα που στοίχισε τη ζωή σε 17 ανθρώπους, με τον πρόεδρο να αναφέρει πως θα ήταν μια καλή ιδέα να εξοπλιστούν οι καθηγητές με πυροβόλα όπλα για να αντιμετωπίζουν τέτοια περιστατικά:

Υπενθυμίζεται πως μέχρι στιγμής, για το 2018 μετράμε ένα περιστατικό πυροβολισμών σε σχολείo κάθε εβδομάδα στις ΗΠΑ, τη στιγμή που οι βιομηχανίες όπλων ρίχνουν σημαντικό μέρος του προϋπολογισμού τους σε προπαγάνδα, προκειμένου να συνεχίσουν τα πυροβόλα όπλα να πωλούνται ελεύθερα στην αγορά των ΗΠΑ.
